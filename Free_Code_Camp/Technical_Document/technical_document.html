Close Git Commit Editor

While using Git, I accidentally used a single apostrophe (') to start and a double apostrophe ("). This created an error on my terminal, and I was not able to exit out of it. There was no error message just a ">" and a cursor. After keyboard mashing, I decided to look it up. Below is an example of my code and the error.

git commit -m 'Add some file to project"
>

Solution

  • Search: "how to get out of commit message git"
  • Site: stackoverflow.com
  • Shortcut: (ctrl x) then (ctrl c)

Offsetting An HTML Anchor

I found that when I clicked on an anchor link, the page would scroll to the wrong part of the section. The top of the section would be cut off by the navbar. This was not a good user experience since the user would need to scroll up to see the titles.

Solution

  • Search: "html anchor not top of section"
  • Site: stackoverflow.com

While I found the answer on the above link, it didn't work correctly for me. I did however, find the code in the comments worked better but was not perfect. Below is the the suggested code and the code I actually used:

Suggested

.thing-with-anchor:before {
  • content: '';
  • display: block;
  • position: relative;
  • width: 0;
  • height: 5em;
  • margin-top: -5em
  • }

Used

.anchor:before {
  • content: '';
  • display: block;
  • height: 75px;
  • margin-top: -75px
  • }

Breakdown why this works

  • .anchor - It is important to place this class on the element that will be called by the link
  • :before - This inserts content before the selected element
  • content: ""; - There needs to be content to occupy space
  • display: block; - This makes the content, inserted by the "before:", push all other sections to a new row.
  • height: 75px: - This is the size of the navbar that is fixed to the top.
  • margin-top: -75px: - Without this in the css, this creates even more space in between the sections.

Center Vertically & Horizontally - Using Flexbox

I had a section of text that I wanted to be center both horizontally and vertically in a div using css. I found a that the flexbox was the best way of doing this.

Solution

  • Search: "css image center in div"
  • Site: W3schools.com

I found the answer on W3schools.com. By using the flexbox, we get more options to align elements using css. While their example worked, I changed it around to work for my specificy example.

Suggested

.center {
  • display: flex;
  • justify-content: center;
  • align-items: center;
  • height: 200px;
  • border: 3px solid green;
  • }

Used

.center {
  • display: flex;
  • justify-content: center;
  • align-items: center;
  • height: 400px;
  • padding-bottom: 75px;
  • }

I ended up using close to the same code as the example with a few changes. Since I was using a nav-bar that was fixed and 75px high, I needed to add padding at the bottom.

Breakdown why this works

  • display: flex; - created a flexbox for the element
  • justify-content: center; - This places the content in the center horizontally
  • align-items: center; - This places the content in the center vertically.
  • height: 400px: - This sets the size of the div and since the iframe I used was 315px it sat nicely in the middle
  • margin-bottom: 75px: - This was used to compensate for the nav-bar offset.
